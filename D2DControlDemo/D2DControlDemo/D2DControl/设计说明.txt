// CD2DSelectUI 设计说明：
// - 控件由触发器（显示区 + 下拉箭头）和下拉列表组成。
// - 触发器使用 CD2DControlUI 作为显示区域，绘制当前选中文本。
// - 下拉箭头在触发器右侧绘制，颜色可配置。
// - 下拉列表为 CD2DListUI 实例，提供选项列表和滚动能力。
// - 样式管理：CD2DSelectStyle 结构体封装触发器样式、箭头颜色、下拉面板样式、选项样式。
//   使用 CAtlMap<ED2DStatus, CD2DSelectStyle> 按状态存储，支持继承默认样式。
// - 显示最大可见项数：通过 SetMaxDropDownItems 设置，内部传递给 CD2DListUI。
// - 交互：点击触发器展开/收起列表；点击选项选中并收起；列表选择变化通过回调通知外部。
